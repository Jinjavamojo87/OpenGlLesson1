Унифицированный, многофункциональный индикатор с полноцветной матричной ЖК-панелью размером 6?8 дюймов. Обеспечивает отображение пилотажной, навигационной, оперативно-тактической (метео-РЛС, TCAS, СРПБЗ) информации и сигнализации, параметров силовой установки, общесамолетного оборудования самолетов и вертолетов гражданской авиации. Имеется несколько вариантов лицевой панели, что позволяет использовать индикатор МФИ-ЖК-2 как для новых, так и модернизируемых воздушных судов.
